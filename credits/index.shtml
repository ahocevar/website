Credits

OpenLayers is thankful to many contributors -- too many to thank individually. This page seeks to list major individuals and organizations who have helped out the OpenLayers project over the years.

Sponsors

If you are interested in sponsoring OpenLayers, please read our sponsorship documents.

Corporate

  • Camptocamp: Supports project with several development resources.
  • DM Solutions: Supports project with several development resources, especially related to proj4js and MapGuide support. Participates within the PSC.
  • Houston Engineering: Funded initial vector layer integration.
  • LISAsoft: Provided initial vector support development. Participates within the PSC.
  • OpenGeo: Supports project with development resources. Has co-sponsored two code sprint events. Participates in PSC.
  • MetaCarta: Founded project, and supports project with continuing hosting and development resources. Has co-sponsored two code sprint events. Participates in PSC.
